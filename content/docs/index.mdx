# Welcome to Konnektr Documentation

Konnektr is a comprehensive platform for building, managing, and scaling **digital twin solutions**. Our modular approach lets you start with what you need and expand as your requirements grow.

## üöÄ Get Started with Konnektr Graph

**Konnektr Graph** is our flagship digital twin runtime - a high-performance, Azure Digital Twins-compatible platform that runs on PostgreSQL with Apache AGE. It's production-ready and available now.

### Why Konnektr Graph?

- **Azure Digital Twins Compatible** - Use standard Azure SDKs (.NET, Python, JavaScript)
- **Built on PostgreSQL + Apache AGE** - Enterprise-grade graph database
- **Real-time Event Streaming** - Stream state changes to Kafka, Azure Data Explorer, and more
- **Powerful Graph Queries** - Complex Cypher queries over your digital twin graph
- **Deploy in Minutes** - Fully managed service or self-hosted

Centralized management for users, organizations, projects, resources, billing, and RBAC across the Konnektr Platform.

**Key Features:**

- Role-based access control (RBAC)
- Billing and subscription management
- Unified resource lifecycle management
- API-first architecture

### DB Query Operator - Kubernetes Operator

Bridge your database state with your Kubernetes cluster. Automatically create, update, and prune resources based on database query results.

**Key Features:**
- Database-driven infrastructure management
- Multi-tenant resource provisioning
- Go templating for flexible manifest generation
- CloudNativePG integration
- Automatic resource pruning

<a href="/docs/db-query-operator" className="text-primary hover:underline font-medium">

### Jexl Extended - Expression Language

Context-based expression parser and evaluator with 80+ built-in functions. Available in JavaScript, Python, and C#.

**Key Features:**
- Identical syntax across all implementations
- 80+ built-in functions (array, string, math, date/time)
- Interactive playground with Monaco Editor
- TypeScript support with IntelliSense
- Custom functions and transforms
