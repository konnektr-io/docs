## Get Started with Konnektr Graph

**Konnektr Graph** is our flagship digital twin runtime - a high-performance, Azure Digital Twins-compatible platform that runs on PostgreSQL with Apache AGE.

<Callout title="Production Ready" type="info">
  Konnektr Graph is available now and ready for production deployments. Deploy a managed instance or self-host in your infrastructure.
</Callout>

### Why Choose Konnektr Graph?

<Cards>
  <Card title="Azure Compatible">
    Use standard Azure Digital Twins SDKs (.NET, Python, JavaScript) without code changes.
  </Card>
  <Card title="PostgreSQL + Apache AGE">
    Built on enterprise-grade graph database with full SQL compatibility and powerful Cypher queries.
  </Card>
  <Card title="Real-time Events">
    Stream state changes to Kafka, Azure Data Explorer, and more with built-in event streaming.
  </Card>
  <Card title="Deploy Anywhere">
    Fully managed service or self-hosted deployment in minutes with Docker or Kubernetes.
  </Card>
</Cards>

## Platform Tools

### KtrlPlane - Cloud Control Plane

Centralized management for the entire Konnektr Platform with multi-tenant support, RBAC, billing, and resource lifecycle management.

<Cards>
  <Card title="Multi-Tenant Management">
    Organizations, projects, and team collaboration with granular permissions.
  </Card>
  <Card title="Role-Based Access Control">
    Fine-grained RBAC for secure access to resources across the platform.
  </Card>
  <Card title="Billing & Subscriptions">
    Usage tracking, billing management, and subscription plans.
  </Card>
  <Card title="API-First Architecture">
    Complete REST API for programmatic resource management and automation.
  </Card>
</Cards>

<Callout title="Explore KtrlPlane" type="info">
  [View KtrlPlane Documentation →](/docs/ktrlplane)
</Callout>

### DB Query Operator - Kubernetes Operator

Bridge your database state with your Kubernetes cluster. Automatically create, update, and prune resources based on database query results.

<Cards>
  <Card title="Database-Driven Infrastructure">
    Automatically provision Kubernetes resources based on database state.
  </Card>
  <Card title="Multi-Tenant Support">
    Isolate resources per tenant with dynamic namespace and resource creation.
  </Card>
  <Card title="Go Templating">
    Flexible manifest generation with Go templates for dynamic resource definitions.
  </Card>
  <Card title="CloudNativePG Integration">
    Native support for PostgreSQL cluster provisioning and management.
  </Card>
</Cards>

<Callout title="Learn More" type="info">
  [View DB Query Operator Documentation →](/docs/db-query-operator)
</Callout>

### Jexl Extended - Expression Language

Context-based expression parser and evaluator with 80+ built-in functions. Identical syntax across JavaScript, Python, and C# implementations.

<Cards>
  <Card title="Cross-Language Support">
    Identical syntax and behavior across JS, Python, and C# implementations.
  </Card>
  <Card title="80+ Built-in Functions">
    Array, string, math, date/time, and more - all with consistent behavior.
  </Card>
  <Card title="Interactive Playground">
    Test expressions with Monaco Editor, IntelliSense, and real-time evaluation.
  </Card>
  <Card title="Custom Extensions">
    Add custom functions and transforms to fit your use case.
  </Card>
</Cards>

<Callout title="Try It Now" type="info">
  [View Jexl Documentation →](/docs/jexl) | [Try Interactive Playground →](https://konnektr-io.github.io/jexl-playground/)
</Callout>
